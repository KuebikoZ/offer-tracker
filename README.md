# Offer Tracker - 求职进度追踪

一个帮助大学生管理求职投递记录的数据分析工具。

## 功能

- 📋 投递列表管理（搜索、筛选、排序）
- 📊 数据可视化分析（行业分布、漏斗图、公司排名）
- 📝 添加投递（手动填写 + 截图识别）
- 🔄 状态流转追踪
- 🎨 橙色主题设计

## 访问地址

**https://your-username.github.io/offer-tracker/**

（部署后替换为你的 GitHub Pages 地址）

## 快速开始

### 本地运行

```bash
# 直接用浏览器打开 index.html
```

### 部署到 GitHub Pages

1. 创建新仓库 `offer-tracker`
2. 上传所有文件到仓库
3. 进入 Settings → Pages → 选择 `main` 分支
4. 访问 `https://your-username.github.io/offer-tracker/`

## 技术栈

- HTML5 + CSS3 + JavaScript
- Tailwind CSS
- Chart.js
- localStorage 本地存储

## 数据说明

- 所有数据存储在浏览器本地（localStorage）
- 更换浏览器或设备后数据不互通
- 清除浏览器缓存会导致数据丢失
